# Debian 8 SSH, Dropbear & OpenVPN Autoscript

### Requirements:
Virtual Private Server (VPS) <br>
Debian 6 (Stable) <br>
Debian 7 (Stable) <br>
Debian 8 (Stable) <br>

### Installation: 

``` apt-get update && apt-get upgrade && wget https://git.io/fhGpL && chmod +x server.sh && ./server.sh```
